Installation
============

### Mac OS X

Please install [Homebrew](http://brew.sh) for Mac before continuing with installation.

```bash
./install.sh
```

### Linux (Ubuntu)

```bash
./install.sh
```

### Windows

*Not supported.*


Usage
=====

To start the [iPython Notebook](http://ipython.org/notebook.html)
you can run:

```bash
./notebook
```
